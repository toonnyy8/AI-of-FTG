# AI-of-FTG
以tensorflow.js實作基於深度學習的FTG AI

## 安裝
```
git clone https://github.com/toonnyy8/AI-of-FTG.git

cd AI-of-FTG

npm i

npm run dev
```

然後在瀏覽器開啟`http://127.0.0.1:8080`

## 建置
```
npm run build
```
開啟./build/index.html
